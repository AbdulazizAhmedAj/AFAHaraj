## Setup
Once you have cloned or downloaded this repository, ensure you have MongoDB installed. Then follow these steps to get the project running:
1. Install the required npm packages:
   ```bash
   npm install
    ```
2.Install bcrypt:
  ```bash
   npm install bcrypt
```
3. Start the server:
    ```bash
   npm start
